Responsive Layout
Easy media queries and customizable breakpoints.
EVA provides three different mixins with responsive media queries:
  • resolutions-above($resolution)
  • resolutions-below($resolution)
  • resolutions-between($min, $max)

Use these mixins under any SCSS rule and use your own rules.

Note: to use these mixins, or any other SCSS funtionality of Eva you need to import the eva-for-scss.scss file. It will not add any bundle size since it includes only sass code without any css classes. 
.some-class { padding: 0.5rem; @include resolutions-above(resolution(large)) { padding: 1rem }};
Change screen size and see this container change colors!
